# midterm\_project group 2



Green Space Accessibility in Graz



This repository contains code and documentation for reproducing a green-space accessibility analysis for the city of Graz.

The workflow follows the methodology from Heikinheimo et al. (2023), adapted to local data availability and implemented entirely with open-source geospatial Python libraries.
